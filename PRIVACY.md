# 隐私政策 / Privacy Policy

**最后更新：2026-07-01**

## 中文

Midea PortaSplit Finder 尊重并保护您的隐私。

### 本扩展不收集任何数据

- ✅ **不收集个人身份信息**（姓名、邮箱、地址等）
- ✅ **不跟踪浏览历史**
- ✅ **不使用 Cookies**
- ✅ **不发送数据到第三方服务器**（仅读取 GitHub 上公开的 JSON 文件）
- ✅ **无后台代码**（Service Worker 未使用）
- ✅ **无需用户登录或注册**

### 数据来源

扩展显示的数据来源于 GitHub Actions 自动爬取的各零售商公开产品页信息，存储于公开的 GitHub 仓库中。所有用户看到的都只是各零售商公开网页上的内容。

### 权限说明

本扩展不需要任何特殊权限（无 `storage`、`alarms`、`notifications`、`host_permissions` 等）。

### 联系方式

如有隐私相关问题，请通过 GitHub Issues 联系我们。

## English

Midea PortaSplit Finder respects and protects your privacy.

### This extension does NOT collect any data

- ✅ **No personal information** (name, email, address, etc.)
- ✅ **No browsing history tracking**
- ✅ **No Cookies**
- ✅ **No data sent to third-party servers** (only reads public JSON from GitHub)
- ✅ **No background code** (no Service Worker)
- ✅ **No login or registration required**

### Data Source

The inventory data shown is automatically scraped from retailers' public product pages via GitHub Actions, stored in a public GitHub repository. Users only see publicly available information from retailer websites.

### Permissions

This extension requires zero special permissions (no `storage`, `alarms`, `notifications`, `host_permissions`, etc.).

### Contact

For privacy concerns, please reach out via GitHub Issues.
